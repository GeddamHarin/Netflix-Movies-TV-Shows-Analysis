# 📊 Netflix Movies & TV Shows Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Netflix's Movies and TV Shows dataset. The dashboard provides insights into content distribution, release trends, ratings, directors, and country-wise availability through dynamic visualizations.

## 🎯 Objectives

* Analyze Netflix content distribution.
* Compare Movies and TV Shows.
* Identify content release trends over the years.
* Explore rating-wise content classification.
* Visualize country-wise content availability.
* Analyze director contributions.

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Netflix Dataset (CSV)

## 📈 Dashboard Features

* KPI Cards for Total Titles, Movies, TV Shows, and Directors
* Movies vs TV Shows Distribution Analysis
* Content Release Trend by Year
* Rating-wise Content Analysis
* Director-wise Analysis
* Country-wise Content Distribution Map
* Interactive Filters and Slicers

## 📊 Key Insights

* Movies make up the majority of Netflix content.
* Netflix content production has increased significantly in recent years.
* Content is distributed across multiple countries worldwide.
* Various rating categories help classify content for different audiences.
* Director analysis highlights major contributors to Netflix's catalog.

## 🚀 How to Use

1. Download the `.pbix` file.
2. Open it using Power BI Desktop.
3. Refresh the dataset if required.
4. Interact with filters and visualizations to explore insights.
